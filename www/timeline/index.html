Timeline

We've been taking care of your eyes for the whole 10 years. Explore a timeline of our journey.

2014

1.0.0 release in July 2014

No user interface, no features. All the extension was doing was injecting a single CSS line:

-webkit-filter: invert(100%);

Later on the extension was able to preserve the colors, which became a killer feature:

-webkit-filter: invert(100%) hue-rotate(180deg);

2015

2.0.0

The app got its user interface.

2018

4.0.0: Dynamic mode

Now the app was able to darken bright page parts, while preserving dark page parts.

2021

Dark Reader for iOS released

The app got configurable color themes.
